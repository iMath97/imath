Auto backup program

Some examples

This script can be used to make backups of almost everything such as your music collection, your photos, important files that you update on a regular basis or even gamesaves.

Today I will show you how to write this script and use it for a Minecraft world.

Setting up

First you need to make a simple text document in a program you like such as Notepad, Notepad++, Sublime text or even something else. I will be using Sublime text.

Then you have to save it as a .bat file (see picture).

Code

After saving the file it is time to code the program. In the next image you can see the code. I will try to explain everything.


Line 1: this will hide the commands in the cmd (not necessary).


Line 2: this will set the output color in the cmd to green (not necessary).


Line 3: this will set the title of your cmd (not necessary).


Line 4 and 5: this creates a variable called 'name' and stores the Minecraft world name and the current date. Change 'WORLDNAME' to the name of your Minecraft world.


Line 6: this will copy the world files to your destination. You need to copy this path. Change 'USERNAME' to your computer username. Change '4ever' to the name of your world. The second path is your destination. You can change this to whatever you like. You just have to keep '\%name%' on the end of the path.


Line 7: this will add a pause on the end of the script so you can see the result.

Running

Your program is ready to run. When you run the program it wil ask you to specify a file or destination. Just press 'D' from directory.
